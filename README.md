# BMI Calculator

A clean, minimal BMI calculator built with vanilla HTML, CSS and JavaScript. No frameworks, no dependencies — just one self-contained HTML file.

## Live Demo

[jakubjalcak1.github.io/bmi-calculator](https://jakubjalcak1.github.io/bmi-calculator)

## Features

- **Metric and imperial** unit support (cm/kg or ft/in/lbs)
- **Sex and age-adjusted categories** — thresholds adapt for children, seniors, and women
- **Animated gauge** — visual indicator that moves smoothly to your result
- **Color-coded result** — underweight, normal, overweight, obese
- **Reference table** — highlights your active category with a colored left border
- **Context notes** — extra info shown for children under 18 and adults over 65
- Fully self-contained single HTML file

## BMI Categories

| Category | BMI Range |
|----------|-----------|
| Underweight | < 18.5 |
| Normal weight | 18.5 – 24.9 |
| Overweight | 25 – 29.9 |
| Obese | ≥ 30 |

> Note: ranges adjust automatically based on entered age and sex.

## Built With

- **HTML** — structure
- **CSS** — styling, animations, responsive layout
- **JavaScript** — BMI logic, dynamic UI updates
